# Code of Conduct

ManaBrew follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## The short version

- Be welcoming. People arrive with different experience levels, languages,
  and backgrounds — in the game and in the code.
- Critique code and ideas, never people. No harassment, no personal attacks,
  no gatekeeping.
- Assume good faith, especially across the language and timezone gaps of an
  open-source project.
- This applies everywhere the project lives: GitHub (issues, PRs, reviews),
  Discord, and in-game interactions in official rooms.

## Enforcement

Report unacceptable behavior to the maintainers — by direct message to the
moderators on [Discord](https://discord.gg/NqrKpbhtcd), or privately through
GitHub to the repository owners. Reports are handled confidentially.
Maintainers may remove content, and warn, kick, or ban anyone who violates
this code of conduct.
