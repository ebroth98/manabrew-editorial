Sent when the engine needs a yes/no decision from the player — for example,
whether to pay an optional cost such as buyback when casting a spell.

## Arguments

`input.type` is `"chooseBoolean"`. The remaining fields:

  <img src="/protocol/choose-boolean.webp" alt="A yes/no prompt with two labelled buttons" />

  </div>

## Response

```ts
type ChooseBooleanOutput = { type: "decision"; value: boolean };
```

Return `true` to accept and `false` to decline.

## Example

```json
{
  "promptId": 7,
  "decidingPlayerId": "player-0",
  "input": {
    "type": "chooseBoolean",
    "presentation": {
      "title": "Buyback",
      "description": "Pay {3} more to return this spell to your hand as it resolves?",
      "targets": []
    },
    "confirmLabel": "Pay",
    "denyLabel": "Decline"
  }
}
```

```json
{ "promptId": 7, "output": { "type": "decision", "value": true } }
```
